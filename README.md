# JTerminal

JTerminal is a simple browser based terminal simulation built with a single HTML file.  
It recreates a minimal command line style interface directly in the browser.

Live Demo  
https://jterminal.netlify.app/

## Overview

This project demonstrates how a terminal style interface can be implemented using basic web technologies. The interface runs entirely in the browser and does not require installation.

The goal of this project is to experiment with terminal UI concepts using lightweight frontend code.

## Features

- Browser based terminal interface
- Command style input
- Lightweight single file implementation
- Minimal design

## Tech Stack

- HTML
- JavaScript
- CSS

## Usage

Open the live demo in your browser:

https://jterminal.netlify.app/

Or download the repository and open `index.html` locally in any browser.
